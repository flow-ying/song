# GitHub Pages 配置步骤

## ✅ 代码已成功推送

你的代码已经成功推送到：`https://github.com/flow-ying/song-tea-game`

## 📋 配置GitHub Pages（最后一步）

### 步骤1：进入仓库设置

1. 访问：https://github.com/flow-ying/song-tea-game
2. 点击仓库顶部的 **Settings**（设置）标签

### 步骤2：配置Pages

1. 在左侧菜单中找到并点击 **Pages**
2. 在 **Source** 部分：
   - **Branch**: 选择 `main`
   - **Folder**: 选择 `/public`
3. 点击 **Save**（保存）

### 步骤3：等待部署

- 等待 1-2 分钟让GitHub部署你的网站
- 刷新页面，你会看到绿色的成功提示和网站地址

## 🌐 访问你的网站

部署成功后，访问：
```
https://flow-ying.github.io/song-tea-game/
```

或者：
```
https://flow-ying.github.io/song-tea-game/index.html
```

## 🔄 后续更新

以后修改代码后，只需：

```bash
git add .
git commit -m "更新说明"
git push
```

GitHub Pages会自动重新部署（通常需要1-2分钟）。

## 📝 注意事项

- 确保图片路径正确：`photos/xxx.png`
- 如果修改了代码，等待1-2分钟让GitHub更新
- 清除浏览器缓存（Ctrl+F5）查看最新版本
